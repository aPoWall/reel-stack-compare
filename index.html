      архитектура
      
        
      Целевая архитектура связывает переносимый iPhone kit, EDL/OTIO state, Remotion/browser preview и agentic render tools. Выбор stack route зависит от цены ошибки и скорости ревью.
      
      
      
      
      
        
      local fast
      ffmpeg + Pillow/OpenCV. Лучший маршрут для одного рилса, быстрых draft renders, проверяемого аудио и простого CI.
      cheaprepeatable
      
        
      state exchange
      OTIO-style model for clips, timing, tracks, markers and metadata. Media stays external; timeline becomes portable.
      EDLinterchange
      
        
      render code
      Remotion: timeline state goes into Player and `inputProps`; server render uses the same JSON-serializable payload.
      ReactrenderMedia
      
        
      browser editor
      WebCodecs/WebGPU class tools prove local preview/export can live in the browser for privacy-sensitive projects.
      OpenReelOmniclip
      
      
      
      
      
        
      
          
      routeusewatch
      
          
            
      Dasha portable coreiPhone ingestion, handoff, fast human/agent templatemacOS-only `avconvert`; account assets separated
      
            
      reel-edit + ffmpegcurrent production MP4s, audio-first, local rendermanual EDL discipline and QA pass required
      
            
      reel-block-edit / Palmiervisual review, draggable blocks, survivor pinningUI can lag behind contracts if state is weak
      
            
      Remotion / OpenCut / ReelForgecomponent overlays, code-driven render, server pipelinestate must be JSON-serializable and stable
      
            
      X-Cut / mcp-video styleagentic operations and chat-driven editing toolstool surface needs clear safety and undo boundaries
      
            
      public dashboard packsharing process, recruiting branches, documenting decisionsprivacy and source attribution gates
      
                
        
      
      
      
    
      

    
      
      
      
        
      qa
      
        
      QA gate is part of the stack. It protects color, audio, privacy, reproducibility and public handoff.
      
      
      
      
      
        
      
          
      preflight
      
          
      • `ffprobe` rotation and color metadata
      • `command -v avconvert` for iPhone pack
      • dependency and asset check
      • raw media outside public repo
      
        
      
        
      
          
      render qa
      
          
      • 1080×1920, fps, yuv420p, bt709 tags
      • AAC stereo 48k, sane bitrate
      • contact sheet: hook, overlay, seam, end
      • listening pass for speech seams
      
        
      
        
      
          
      public firewall
      
          
      • no real names, handles, filenames, paths
      • no raw transcript or screenshots
      • public metrics are aggregated
      • recipes use `creator_a`, `hook`, `handoff`
