# Article Reference

Use this public artifact as:

> **Reels Pipeline OS** is an open visual map for editing short-form videos with an agent-assisted workflow. It describes the full path from raw recording to final vertical MP4: source audit, transcript with word timings, source blocks, EDL as the editing source of truth, overlay lanes, render route, QA and public-safe branch packs.

Short link:

https://apowall.github.io/reels-pipeline-os/

## One-Paragraph Description

The project turns Reels editing into a reusable pipeline with branch packs, repeatable state and public-safe examples. A creator or agent can choose a source type, goal and editing mode, then generate expected artifacts, a JSON config and a prompt. Public examples use generic timings, labels and overlay lanes so the method can be shared without exposing footage, transcripts, private names or local project paths.
